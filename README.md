# Παιχνίδι Περιοδικού Πίνακα (Α’ Λυκείου)

Αυτό το repository είναι δημόσιο **μόνο για σκοπούς προβολής του εκπαιδευτικού σεναρίου**.  
**Ο κώδικας δεν επιτρέπεται να αντιγραφεί, να τροποποιηθεί ή να αναδιανεμηθεί χωρίς άδεια του δημιουργού.**

## Περιγραφή
Διαδραστικό παιχνίδι για την Α’ Λυκείου:
- Στοιχεία με Z από 3 έως 20
- Drag-and-drop για ταξινόμηση ανά αριθμό ηλεκτρονίων σθένους
- Συμπλήρωση ηλεκτρονιακής δομής
- Μήνυμα για σύγκριση με τον Περιοδικό Πίνακα
